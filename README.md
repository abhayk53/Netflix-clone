# 🎬 Netflix Clone (HTML & CSS)

A responsive front-end clone of the Netflix landing page built using pure HTML and CSS. This project replicates the UI design and layout of Netflix for learning and practice purposes.

---

## 🚀 Features

- 🎨 Clean and modern UI inspired by Netflix
- 📱 Responsive design (mobile + desktop)
- 🎥 Embedded video previews
- 📂 Structured layout with sections
- ❓ FAQ section with interactive hover effects

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox, Grid)
- Google Fonts

---

---

## ▶️ How to Run

1. Download or clone the repository:
```bash
git clone https://github.com/abhayk53/netflix-clone.git
Open INdex.html in your browser
```
